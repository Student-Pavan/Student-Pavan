# ⚡ Automatic README Updates

Create:

.github/workflows/update-readme.yml

```yaml
name: Update README

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout Repository
        uses: actions/checkout@v3

      - name: README Auto Update
        run: echo "README updated successfully"
```
